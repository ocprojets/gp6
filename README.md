# Organisation du github
Les fichiers software et hardware ne contienne que les fichiers html nécessaire au site internet. Le dossier 'img' lui contient que des images que vous retrouverez sur le site. Le document index.html qui ne se trouve ni dans hardware ou software est la page d'acceuil sur laquelle vous tombez si vous utilisé le site https://ocprojets.github.io/gp6/.
Vous trouverez tous les codes sous le dossier intitulé code. Les seuls fichiers qui sont intéressant et que nous mentionnons dans notre site sont les fichiers suivants: `clapbip.py`, `mic-File.py`, `graph-mic.py` et `micV1`. Les autres n'étaient que des tests ou à des buts de débuggage.
Noter que le site internet à été effectué sans l'aide quelconque d'une IA (tout, du CSS à chaque fichier html).
